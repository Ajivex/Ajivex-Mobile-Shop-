# 🚀 Ajivex Mobile Shop — GitHub Pages Deploy Guide

## Files Created (Phase 1)
- `index.html` — Full website (all pages)
- `style.css` — Master stylesheet
- `app.js` — All JavaScript logic
- `manifest.json` — PWA manifest

---

## Step 1: Create GitHub Account
Go to https://github.com → Sign Up (free)

---

## Step 2: Create New Repository
1. Click "+" → "New repository"
2. Name: `ajivex-mobile-shop`
3. Set to **Public**
4. Click "Create repository"

---

## Step 3: Upload Files
1. Click "Add file" → "Upload files"
2. Drag and drop all 4 files:
   - `index.html`
   - `style.css`  
   - `app.js`
   - `manifest.json`
3. Click "Commit changes"

---

## Step 4: Enable GitHub Pages
1. Go to repository → "Settings"
2. Left sidebar → "Pages"
3. Source: "Deploy from a branch"
4. Branch: `main` → folder: `/ (root)`
5. Click "Save"

---

## Step 5: Your Site is LIVE! 🎉
URL: `https://YOUR-USERNAME.github.io/ajivex-mobile-shop`

Takes 2-5 minutes to deploy first time.

---

## Admin Portal Access
1. On website, click anywhere that goes to Admin
2. Or directly: `navigate('admin')` in browser console
3. Username: `ajivex_admin`
4. Password: `Admin@2025`
